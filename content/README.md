# Website Content

Edit `site-content.js` when you want to update page text, links, images, or publications.

The main files are:

- `site-content.js`: editable content for About, Music, MEXT, and Research.
- `../index.html`: page shell only. You usually do not need to edit this.
- `../styles/site.css`: visual design and layout.
- `../scripts/site.js`: builds the page from `site-content.js`.

Use paths relative to the project root, for example `images/profile.jpeg` or `files/paper.pdf`.
